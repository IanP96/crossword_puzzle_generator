# Crossword cryptic puzzle generator

Inspired by the game from the newspaper that I forgot the name of.

## How to play

1. Run `make run`
2. Find the puzzles and solutions in the `output` directory

## Difficulty

Change the difficulty by changing the value of `BASE_THRESHOLD` in `main.c`.
